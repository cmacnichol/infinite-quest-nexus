Full-reload compression details

Automatic chooses the richest format that fits. Full prefers verbatim accepted turns. Detailed shortens older turns but keeps scene continuity. Compact favors chronological coverage.

▸ Advanced Start Features

Optional opening tools for stories that need a player-facing prologue or a custom first action.

▸ Default trackers

Define the default world-state trackers that appear in Edit State → Trackers. When a new adventure starts, the current state is reset from these triggers and their starting values.

▸ Event triggers

When enabled, custom event triggers stay saved but the app skips all trigger-check LLM prompts and behaves as though no event triggers exist.

Define special event triggers that the app checks with a separate LLM prompt before or after the main story response. Activated trigger instructions can influence the next story response, or optionally add a brief after-response passage.

▸ RPG d% statistics

When enabled, direct browser mode resolves the action locally; Nexus mode persists the private assessment and d100 result in the durable backend job. In both modes, the story model receives only the fictional consequence—never the roll, stat, target, modifier, or difficulty.
