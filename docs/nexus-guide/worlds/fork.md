# Fork a world version

Forking copies one immutable version into a new independent editable draft.

1. Select the source world and **Published version**.
2. Open **More actions**.
3. Select **Fork selected version**.
4. Enter a **Fork title**.
5. Select **Fork world**.

The fork records provenance but has its own identity, draft revision, publication history, and campaigns. Later source-world changes do not update the fork, and fork changes do not update the source.
