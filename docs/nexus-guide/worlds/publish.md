# Publish a world version

Publishing freezes the current draft into an immutable numbered snapshot.

1. Save the latest draft.
2. Confirm that the playable-character roster is campaign-ready.
3. Select **Publish version**.
4. Confirm publication.

The new entry appears under **Published version**. Publication does not edit earlier versions or move existing campaigns.

Release notes describe this snapshot. Revise the draft and publish another version when canon changes; do not attempt to edit a published snapshot in place.
