# Archive or restore a world

Archiving removes a world from active authoring without deleting its history.

1. Select the world.
2. Open **More actions**.
3. Select **Archive**.

Archived worlds cannot be edited or published. Existing campaigns remain available because they depend on immutable versions, not on the draft's active status.

To resume authoring, select the archived world, open **More actions**, and select **Restore**.
