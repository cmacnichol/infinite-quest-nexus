# Select and load a campaign

On the dashboard, use campaign search or the campaign carousel and select a card to resume at the latest accepted scene. If a durable generation is already pending, the Story page reconnects and resumes progress.

For advanced review, open **Setup → Campaign Management**, then select a campaign from the list. Review its title, status, pinned version, text profile, and story length before using its story-opening action.

A campaign created with the basic dashboard form opens immediately. A zero-turn campaign displays its background when available and queues the world opening action. Configure and test the default text profile before creating or opening it for the first time.
