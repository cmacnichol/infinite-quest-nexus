# Select and load a campaign

Select **Campaigns**, then select a campaign from the list. Review its title, status, pinned version, text profile, and story length.

Select **Load story** to open the campaign-specific Infinite Quest route. Nexus prepares the accepted history and reconnects the player to any pending durable generation.

A zero-turn campaign immediately displays its background when available and queues the world opening action. Configure and test its text profile before loading it for the first time.
