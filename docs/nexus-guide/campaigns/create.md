# Create a campaign

1. Select a world and an exact **Published version**.
2. Open **More actions** and select **Create campaign**.
3. Enter a **New campaign title**.
4. Choose the **Campaign player character**.
5. Choose a **Turn control style**:
   - **Player actions only** for mechanics-mediated attempts.
   - **Flexible — Auto** to classify each free-form submission initially.
   - **Flexible — Action first** or **Flexible — Scene direction first** to expose all modes with an explicit initial selection.
6. Select **Create campaign**.

The initial control-style selection comes from the current user's **Default turn input style for new campaigns** profile preference. You can override it for this campaign without changing that preference.

Campaign creation is disabled until the selected version has a complete playable-character roster. The chosen character and world version are snapshotted into the campaign so later world edits cannot alter the story silently.

The control style affects future turns, not world canon. Generated choices and the opening action always remain Action. See [Turn input modes](../../player-guide/turn-input-modes.md).
