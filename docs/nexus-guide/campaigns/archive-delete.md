# Archive or delete a campaign

## Archive

Set **Status** to **Archived** and select **Save campaign**. Archiving keeps the accepted ledger, state, Chronicle, assets, and audit records while removing the campaign from ordinary active play.

Set the status back to **Active** to resume.

## Delete

::: danger Permanent deletion
Campaign deletion removes its accepted turns, campaign state, Chronicle memory, jobs, and asset records. Export important story data and verify an operator backup first.
:::

1. Select **Delete campaign**.
2. Type the exact campaign title.
3. Confirm deletion.

Deleting a campaign can release a world-version dependency, but it does not delete the world automatically. Provider cost audit semantics may require retained operational records according to the current schema.
