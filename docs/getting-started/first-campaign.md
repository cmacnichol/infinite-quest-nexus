# Create your first campaign

A campaign is a persistent story instance created from one exact published world version.

## Create from World Library

1. Select the intended world and choose its **Published version**.
2. Open **More actions**.
3. Select **Create campaign**.
4. Enter a **New campaign title**.
5. Choose the **Campaign player character**.
6. Select **Create campaign**.

Nexus copies the selected character into the campaign. Later changes to the world roster or a newly published version do not silently change that campaign snapshot.

## Review campaign settings

Select the new campaign in **Campaigns**. Its management panel provides:

- **Campaign title**
- **Status**
- **World version**
- **Text provider profile**
- **Default story response length**

Confirm that the text provider is enabled and has a model. The story response length is a campaign preference; the provider's maximum output remains a separate safety ceiling.

Select **Save campaign** after changing settings.

## Load Infinite Quest

Select **Load story**. Nexus opens the campaign-specific Infinite Quest route and hands off the server-generated accepted history without placing provider credentials in browser storage.

For a new campaign, Infinite Quest shows the background story when available and automatically submits the world's opening action. Configure the provider before selecting **Load story** so this first generation can start successfully.

Continue with [Generate your first story turn](./first-story-turn.md).
