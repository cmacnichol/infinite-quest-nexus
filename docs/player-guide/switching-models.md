# Switch text models

Model selection belongs to a saved text-provider profile and a campaign's provider selection.

## Change the model on a profile

1. Open **Setup → Provider Setup**.
2. Edit the applicable **Story text** profile.
3. Refresh or open the model picker.
4. Select the intended default model.
5. Review its context window and maximum output.
6. Select **Save provider**.

## Change the campaign provider

1. Open **Setup → Campaign Management** and select the campaign.
2. Select a different **Text provider profile**.
3. Select **Save campaign**.

The next generation bootstraps from authoritative database state and selected Chronicle context. Provider continuation identifiers are recovery optimizations only; switching a model does not abandon the campaign ledger or cause one campaign's chain to be reused in another.
